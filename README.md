### Todo-List

### 介绍

一个简单美观的 Todo-List 应用，使用了 React 框架，实现添加 todo ，删除 todo ，标记 todo 等基本功能，使用 CSS3 中的定位（flex ，absolute）、阴影等属性使界面美观，还利用 LeanCloud 提供的数据存储功能，使用 JavaScript 完成用户注册、登录、重置密码功能。
